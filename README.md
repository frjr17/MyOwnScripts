## Fedora Setup
Keyboard Commands
  - Settings - Super + I
  - Move window one monitor down  - Ctrl + Alt + Down
  - Move window one monitor Left  - Ctrl + Alt + Left
  - Move window one monitor Right  - Ctrl + Alt + Right
  - Move window one monitor Up  - Ctrl + Alt + Up
  - Move window one workspace to the left - Shift + Super + Left
  - Move window one workspace to the right - Shift + Super + Right
  - Open the quick settings menu - Super + A
  - Show the notification list - Super + N
  - Toggle fullscreen mode - F4
  - Open Nautilus (Custom Shorcut) -  Super + E
